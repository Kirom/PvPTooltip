# PvP Tooltip - Enhanced Player Information

**Instantly see comprehensive PvP information for any player you encounter in World of Warcraft!**

PvP Tooltip is a lightweight, performance-optimized addon that enhances your gameplay by displaying detailed PvP statistics directly in player tooltips. Whether you're evaluating potential teammates, assessing opponents, or simply curious about other players' PvP experience, this addon provides all the information you need at a glance.

## ✨ Key Features

### 🎯 **Comprehensive PvP Data Display**
- **Current Season Ratings** for all competitive brackets (2v2, 3v3, Shuffle, RBG, Blitz)
- **Personal Best Ratings** showing historical peak performance
- **Season Statistics** including games played and win rates
- **Color-coded ratings** for instant skill assessment

### 🌍 **Multi-Region Support**
- Supports both **EU and US regions** with daily-updated databases
- Automatic region detection and appropriate data source selection
- Cross-realm and cross-faction player lookup capabilities

### 🎨 **Clean, Intuitive Design**
- **Professional color coding**: White (0-1799), Green (1800-2099), Blue (2100-2399), Purple (2400+)
- **Win rate indicators**: Red (≤50%), Green (>50%)
- **Organized layout** with clear sections for easy reading
- **Minimal performance impact** with optimized data structures

### 🔧 **Zero Configuration Required**
- **Install and play** - no setup needed
- **Automatic integration** with existing tooltip systems
- **Compatible** with popular UI addons and frameworks

## 📊 What You'll See

When hovering over any player, the addon adds a "PvP Tooltip info" section displaying:

### Current Rating
View current season ratings across all competitive brackets with appropriate color coding based on skill level.

### Character Experience  
See personal best ratings to understand a player's historical peak performance and overall PvP experience.

### Current Season Statistics
Check games played and win rates to assess current activity level and performance trends.

## 🎮 Where It Works
