# PvP Tooltip - Enhanced Player Information

**Instantly see comprehensive PvP information for any player you encounter in World of Warcraft!**

PvP Tooltip is a lightweight addon that enhances your gameplay by displaying detailed PvP statistics directly in player tooltips. Whether you're evaluating potential teammates, assessing opponents, or simply curious about other players' PvP experience, this addon provides all the information you need at a glance — no clicks, no website, no setup.

## ✨ Key Features

### 🎯 **Comprehensive PvP Data Display**
- **Current Season Ratings** for all competitive brackets (2v2, 3v3, Solo Shuffle, RBG, Blitz)
- **Personal Best Ratings** showing all-time peak performance per bracket
- **Season Statistics** including games played and win rate
- **Per-spec breakdown** for Solo Shuffle and Blitz, with the hovered player's active spec highlighted
- **Color-coded ratings** for instant skill assessment

### 🌍 **Multi-Region, Frequently Refreshed**
- Region databases refreshed frequently from current ranked data
- Automatic region detection and appropriate data source selection
- Cross-realm and cross-faction player lookup

### 🎨 **Clean, Intuitive Design**
- **Color coding**: White (0-1799), Green (1800-2099), Blue (2100-2399), Purple (2400+)
- **Win rate indicators**: Red (≤50%), Green (>50%)
- **Organized layout** with clear sections for easy reading
- **Minimal performance impact**, no measurable overhead

### ⚙️ **In-Game Settings Panel**
- Open with `/pvpt` or `Esc → Options → AddOns → PvPTooltip`
- **Live tooltip preview** updates as you change options
- Toggle sections, brackets and a show-on-modifier key (Shift/Ctrl/Alt)
- Choose all specs vs only the hovered spec; hide brackets with no games

### 🔧 **Zero Configuration Required**
- **Install and play** — sensible defaults out of the box
- **Automatic integration** with existing tooltip systems
- **Compatible** with popular UI addons and frameworks

## 📊 What You'll See

![PvP Tooltip example](https://raw.githubusercontent.com/Kirom/PvPTooltip/master/assets/tooltip-example.png)

Hovering any player adds these sections to the tooltip:

### Current Rating
Current-season rating for each enabled bracket, color-coded by skill level.

### Character Experience
All-time personal-best rating per bracket.

### Current Season
Games played and win rate per bracket for the current season.

## 🔄 Installation & Updates

1. Install like any WoW addon (CurseForge, Wago, WowUp, or manual)
2. Restart your game or `/reload`
3. **Keep it updated** (enable auto-update) for the freshest data
4. Enjoy enhanced player tooltips immediately!
